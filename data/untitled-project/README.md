# Untitled Project Data Lane

Temporary data surface for the next project before its workflow name is locked.

Use this folder only for exploratory schemas, sample inputs, source notes, or
fixtures that do not belong to an existing lane.

Promotion checklist:

- repeated real-world job is named;
- input contract is clear;
- output artifact is clear;
- safety boundary is written;
- final project slug is chosen;
- files are moved to `data/<final-project-slug>/`.
