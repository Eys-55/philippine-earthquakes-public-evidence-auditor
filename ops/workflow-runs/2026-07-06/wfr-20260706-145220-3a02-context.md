## Premise Lock

Create a new workflow for monitoring the control repo tracker with a local,
read-only Astro dashboard. The dashboard is a view over tracker authority, not a
replacement for tracker authority.

## Workflow Contract

- repeated job: render current tracker state as an operator dashboard;
- input: local tracker JSON and upload-state files;
- output: static Astro monitor plus exported JSON snapshot;
- lanes: tracker-data, UI, validation, review;
- validation: exporter test, Astro build, tracker validation, diff check;
- safety: no external writes, no credentials, no mutation from the browser, no
  3D or canvas monitor surface.
