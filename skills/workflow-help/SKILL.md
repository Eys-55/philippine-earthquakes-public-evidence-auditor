# Workflow Help

Show the operator the workflow slash surface without starting a phase.

## Commands

- `/workflow-router` - route any workflow, bug, project edit, continuation, or
  phase request.
- `/workflow-intake` - explicitly create or lock workflow state.
- `/workflow-grilling` - ask one context-aware grilling question.
- `/workflow-to-prd` - move agreed context into PRD.
- `/workflow-to-issues` - split an approved PRD or plan into vertical slices.
- `/workflow-implement` - implement approved scope.
- `/workflow-code-review` - review against spec and standards.
- `/workflow-closeout` - validate, close tracker state, commit, push, and
  verify upload.
- `/tracker-workflow` - tracker-specific workflow intake alias.
- `/tracker-status` - show tracker status.
- `/tracker-closeout` - close tracker-backed work.
- `/workflow-help` - show this list.

If tracker status makes one next command obvious, recommend it.
