# Workflow Code Review

This skill wraps the Matt Pocock `code-review` phase.

## Required Flow

1. Read `.agents/skills/code-review/SKILL.md`.
2. Checkpoint the run to `current_skill=code-review`.
3. Review against standards and spec as separate axes.
4. Lead with findings and file references.

Do not bury findings under a summary. Ask for a fixed point when required.
